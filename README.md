# Karen Tatiana Martínez — Personal Portfolio

A Netflix-inspired personal portfolio website built with vanilla HTML, CSS, and JavaScript.

🌐 **Live Site:** [katamar-lang.github.io/portfolio](https://katamar-lang.github.io/portfolio)

## ✨ Features

- **Netflix-style UI** — dark theme, red accents, cinematic feel
- **Splash screen** with loading animation
- **Hero section** with animated entrance
- **About section** with card-based layout
- **Animated skill bars** triggered on scroll
- **Projects showcase** with hover previews (Netflix card style)
- **Stats counter** animation on scroll
- **Contact form** with validation
- **Fully responsive** — mobile, tablet, desktop
- **Smooth scroll reveal** animations throughout

## 🛠 Technologies

- HTML5 (semantic markup)
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (DOM manipulation, Intersection Observer pattern, scroll events)
- Google Fonts (Bebas Neue + DM Sans)
